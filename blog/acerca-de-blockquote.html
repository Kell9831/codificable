Uso de blockquote

El elemento HTML <blockquote>, que se usa para citar contenido proveniente de otra fuente. El texto del bloque puede ser formateado con la etiqueta <cite>. La etiqueta <q> es usada cuando el autor no quiere darle formato al texto. En este caso, solo aparecerá una línea en blanco antes y después del citado.

<blockquote cite=""url-de-la-fuente"">
    <!-- Cita -->
    </blockquote>

blockquote ⬇️

Nota. Recomendamos que las implementaciones de hojas de estilo porporcionen un mecanismo para insertar signos de puntuación de citas antes y después de una cita delimitada por un BLOCKQUOTE de un modo apropiado según el contexto del idioma actual y el grado de anidamiento de las citas.

Descripcion del codigo

El código comienza con las etiquetas estándar <html>, <head>, <tittle>, etc. En la etiqueta <body> se da el <header> en la cabecera <h1>.
Se inicia la etiqueta <blockquote>. Todo el texto se separa del otro texto por esta etiqueta como se muestra en la salida. Nunca olvides cerrar las etiquetas.

La etiqueta <blockquote> puede contener uno o más elementos en bloque o script, como:

  • párrafos
  • encabezados
  • listas
  • imágenes
  • etc.

Etiqueta <q>

La etiqueta <blokquote> tiene un hermano menor: la etiqueta <q>, que sirve para citas en línea, es decir, dentro de un párrafo o texto normal. A diferencia de <blockquote>, la etiqueta <q> sí entrecomilla los contenidos automáticamente, según el idioma y el nivel de anidamiento.

La etiqueta `<q>` en HTML se utiliza para indicar una cita corta en línea. Esta etiqueta se utiliza para citar texto de otra fuente dentro de un párrafo. Los navegadores suelen agregar comillas antes y después del contenido de la etiqueta `<q>`. Por ejemplo, si tienes un miniblog y quieres citar una frase famosa o una línea de un libro, puedes usar la etiqueta `<q>` para hacerlo. Aquí hay un ejemplo:```html <p>Como dijo Albert Einstein: <q>La imaginación es más importante que el conocimiento.</q></p>```Esto se mostraría en el navegador como: <p>Como dijo Albert Einstein: "La imaginación es más importante que el conocimiento."</p>

Diferencias entre <blockquote> y <pre>

  1. La etiqueta <blockquote> se utiliza para crear un párrafo citado que se desplaza del texto regular. Se crea un espacio y aparece como una entidad diferente. El resultado se puede ver en las imágenes.
  2. La etiqueta <pre> se utiliza para preservar el formato del texto tal como está escrito en el código. No elimina ningún salto de línea ni espacio. El resultado se puede ver en las imágenes.
  3. La etiqueta <blockquote> puede contener uno o más elementos en bloque o script, como párrafos, encabezados, listas, imágenes, etc.
  4. La etiqueta <pre> solo puede contener texto o elementos en línea, como enlaces, imágenes, etc.
  5. La etiqueta <blockquote> se usa para mostrar texto de otra fuente, como una cita de un libro o un artículo. La etiqueta <pre> se usa para mostrar código o texto preformateado, como un fragmento de código o un poema.
  6. La etiqueta <blockquote> suele estar acompañada de un elemento que indica la fuente de la cita. La etiqueta <pre> no tiene ningún elemento asociado que indique la fuente del texto.
